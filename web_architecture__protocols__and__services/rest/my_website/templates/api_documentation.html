API Dokumentacija

  • GET /posts - Dohvati postove
  • POST /posts - Kreiraj novi post
  • GET /users - Dohvati sve korisnike
  • GET /users/<user_id> - Dohvati detalje korisnika
  • PUT /users/<user_id> - Ažuriraj korisnika
  • DELETE /users/<user_id> - Izbriši korisnika
  • GET /users/<user_id>/posts - Dohvati sve postove određenog korisnika
  • GET /users/<user_id>/posts/<post_id> - Dohvati detalje posta
  • PUT /users/<user_id>/posts/<post_id> - Ažuriraj post
  • DELETE /users/<user_id>/posts/<post_id> - Izbriši post
